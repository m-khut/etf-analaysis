This project analyzes performance and risk characteristics of major ETFs using **Python, Pandas, NumPy, and Matplotlib**.

It currently supports:
- SPY — S&P 500
- QQQ — Nasdaq 100
- TLT — Long-Term US Treasuries
- GLD — Gold
- USO — Crude Oil

The project is designed to help understand:
- Returns behavior
- Portfolio performance
- Risk and diversification
- Market behavior across years & months
- Calculate Moving Average (MA 50)
- Calculate and compare returns based on 50 MA vs Buy & Hold strategies

Features

ETF Level Analysis
- Daily returns
- Annual returns table
- Monthly returns table
- Cumulative growth curves

Portfolio Level
- Equal-weight portfolio returns
- Portfolio cumulative growth
- Correlation matrix
- Price trends comparison

Strategy
 - Calculate and return strategy comoarision (MA vs Buy&Hold)
